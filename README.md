# Personal Health Tracker

Personal Health Tracker is a frontend-based web application that helps
users monitor daily health activities such as BMI, water intake, sleep,
and calorie tracking.

## Features
- BMI Calculator
- Water Intake Tracker
- Sleep Tracker
- Calorie Tracker
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Project Type
Frontend-based Web Application
